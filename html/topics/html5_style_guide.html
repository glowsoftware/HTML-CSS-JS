HTML5 Style Guide

  1. Always keep your code tidy, clean, and well-formed.

  2. Always declare <!DOCTYPE html> as the first line in your document.

  3. Use lower case element names.

  4. Close all HTML elements (also empty elements).

  5. Use lower case attribute names.

  6. Quote attribute values.

  7. Always add the "alt" attribute to images.

  8. Avoid long code lines.

  9. Do not add blank lines without a reason.For readability, add blank lines to separate large or logical code blocks.

  10. Do not omit the <html>, <head> and <body> tags.

  11. The <title> element is required in HTML5.

  12. Use simple syntax for linking to style sheets (the type attribute is not necessary).

  13. Short CSS rules can be written compressed, on one line. Long rules should be written over multiple lines.

  14. HTML files should have a .html extension. CSS files should have a .css extension. JavaScript files should have a .js extension.
