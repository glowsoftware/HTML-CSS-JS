Attribute Selectors

It is possible to style HTML elements that have specific attributes or attribute values.

CSS [attribute] Selector

Image with an alt attribute gets a yellow border:

CSS [attribute="value"] Selector

Image with alt="Northern Lights" gets a yellow border:

CSS [attribute~="value"] Selector

Image with alt="Trolltunga Norway" gets a yellow border, because alt value contains word 'Norway':

CSS [attribute|="value"] Selector

Image with alt="Mountains" gets a yellow border, because alt value starts with word 'Mountains':

Note: The value has to be a whole word, either alone or followed by a hyphen( - )!

CSS [attribute^="value"] Selector

Image with alt="Trolltunga Norway" gets a yellow border, because alt value begins with 'Troll':

Note: The value does not have to be a whole word!

CSS [attribute$="value"] Selector

Images with alt="Northern Lights" and alt="Mountains" gets a yellow border, because alt value ends with 's':

Note: The value does not have to be a whole word!

CSS [attribute*="value"] Selector

All images gets a yellow border, because all have alt value which contains 'o':

Note: The value does not have to be a whole word!
